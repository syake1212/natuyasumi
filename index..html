宇宙を舞台とした壮大な旅。美麗なアニメ表現と軽快なテンポが魅力。

2. 世界観・ストーリー

プレイヤーは「開拓者（Trailblazer）」として、星穹列車（Astral Express）の一員に。 銀河に散らばる星々を巡り、謎の存在「星核（Stellaron）」に関わる事件を解決していく。 「崩壊」シリーズ世界の別線上の物語で、親しみつつも新鮮な冒険が楽しめる。

宇宙旅 星核 星穹列車 銀河鉄道

3. 主要キャラクター

開拓者

プレイヤーの分身。性別選択可。属性切替で役割も変化。

三月なのか

明るいムードメーカー。矢とサポートで戦況を整える。

丹恒

冷静な槍使い。氷属性を得意とするストイックな青年。

姫子 / ヴェルト

頼れる研究者と知識人。シリーズの縁も感じられる。

4. ゲームシステム

ターン制コマンド

通常攻撃 / 戦技 / 必殺技を使い分け、弱点撃破で主導権を握る。

育成と装備

レベル / 昇進、光円錐、遺物、星魂で多面的に強化。

惑星探索

星ごとに宝箱・謎解き・ミッション。広めのフィールドを冒険。

5. ガチャ（跳躍）

  • 種類：キャラ／光円錐
  • レア：★5が最高レア（天井あり）
  • 所感：無料配布やイベントである程度引けるが、限定狙いは計画的に。

6. イベント・アップデート

およそ6週間サイクルで大型アップデート。新ストーリー／新キャラ／新イベントが順次追加。 期間限定イベントの報酬が大きいので、定期的なログインが吉。

7. 崩壊シリーズとの関係

「崩壊3rd」と同じシリーズに属するが、直接の続編ではない別線の物語。 姫子やヴェルトなど共通キャラの登場で、シリーズファンもニヤリ。

8. 評価・魅力

表現美

アニメ調×高品質。演出・カメラワークが映える。

物語性

壮大な宇宙譚にユーモア。キャラ会話の掛け合いも良。

遊びやすさ

テンポ良いターン制。オート／倍速でサクサク。
